# 贡献说明

欢迎改进这个 Skill。

这个仓库主要维护一套中文社媒文案写作规则，重点是让内容更像真人写的经验分享，而不是模板感很重的 AI 文案。

## 可以贡献什么

- 补充更自然的中文表达规则
- 增加更实用的社媒文案模板
- 优化 crypto、空投、工具教程类内容的写法
- 删除容易显得空泛、夸张、套路化的表达
- 修正文案里的错别字、表述不清或重复内容

## 修改原则

- 保持简体中文。
- 语言要朴素，不要堆漂亮词。
- 不要加入“互推、助推、找人转发”这类分发规则。
- 不要加入夸张、绝对化、制造焦虑的表达。
- 修改后尽量检查一遍禁用词和括号。

## 文件说明

- `human-social-copy/SKILL.md` 是核心规则。
- `human-social-copy/agents/openai.yaml` 是技能展示信息。
- `human-social-copy/references/patterns.md` 是模板和示例。

## 提交建议

提交信息可以用中文，比如：

```text
优化中文写作规则
补充空投文案模板
修正 README 说明
```
