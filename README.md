# 真人感社媒文案 Skill

这是一个用于中文社媒文案写作的 Codex Skill。

它适合把草稿、笔记、截图内容改成更像真人写的内容，重点是：

- 钩子加干货加 CTA
- 语言朴素、真实、可信
- 少标题，不写线程，清理括号
- 避免 AI 腔、空话、夸张表达
- 选题优先考虑对读者有用和贴近当下热度

## 使用方式

把 `human-social-copy` 文件夹放到 Codex skills 目录后，可以这样调用：

```text
使用 $human-social-copy，把这段草稿改成钩子加干货加 CTA 的中文社媒文案。
```

## 内容结构

```text
human-social-copy/
  SKILL.md
  agents/openai.yaml
  references/patterns.md
```
