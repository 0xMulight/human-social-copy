# 自适应钩子参考

> **没有钩子，寸步难行。** 钩子不是锦上添花——没有钩子，内容再好也等于零曝光。第一步永远是想钩子，第二步才是填内容。

钩子是决定帖子生死的第一行。不要套固定模板——先用8大模型判断这条内容最适合哪种钩子，再从模板库里匹配具体句式。

## 快速决策：选哪个模型

拿到内容后，先回答一个问题：**这条内容最强的情绪引擎是什么？**

| 内容特征 | 选这个模型 |
|---|---|
| 有反直觉发现/秘密/内幕 | 好奇模型 |
| 解决一个让人烦的问题 | 痛点模型 |
| 有before/after对比 | 反差模型 |
| 只对特定人群有用 | 圈定受众模型 |
| 涉及"落后/被淘汰"焦虑 | 焦虑模型 |
| 涉及风险/损失/踩坑 | 恐惧模型 |
| 有独家数据/方法/干货 | 高价值模型 |
| 可以蹭名人/热点/趋势 | 借势模型 |
| 挑战普遍认知，打破"XX只能YY"的假设 | 反常识模型 |

## 9大钩子核心模型

### 1. 好奇模型 —— 只说一半的秘密 + 关键留白

原理：激发"到底是什么"的探索欲，不读完难受。

公式：`[反常现象/秘密预告] + [暗示答案但不给全]`

示例：
- `让优质内容持续被转发的核心，不是文笔…`
- `我发现了让AI写中文不再像机翻的诀窍，就改了一个地方…`
- `这个工具处理图片的方式，跟所有同类都不一样…`

AI/工具/Crypto 场景适配：
- `这个开源工具把AI写作查重压缩到了极致，方式有点暴力…`
- `跑了一圈AI Agent交易比赛，发现最赚钱的策略反而是最简单的…`

### 2. 痛点模型 —— 精准扎心困境 + 立刻给方案

原理：让读者觉得"说的就是我"，然后给出口。

公式：`[你也在经历的困境]？[我有解法]`

示例：
- `AI写的推文一眼就能看出来？问题出在这3类句式上。`
- `同样的prompt，别人出图精美你出图糊？差的不是模型…`
- `花几千买AI工具，结果吃灰？先看这3个判断标准。`

AI/工具/Crypto 场景适配：
- `装了十几个AI插件，真正每天在用的就这2个。`
- `空投撸了一堆账号，最后只活了3个？你踩了女巫检测的坑。`

### 3. 反差模型 —— 过去糟糕状况 + 现在美好结果

原理：对比产生冲击，让人想知道"怎么做到的"。

公式：`[垃圾状态] → [翻身结果]，[关键转折只提一句]`

示例：
- `AI写的文案从"一眼假"到朋友以为是我自己写的，就改了一套规则。`
- `之前每次发推像扔石头进大海，改了第一行之后互动翻了10倍。`
- `体脂率30%到18%，我没节食，全靠这4个习惯…`

AI/工具/Crypto 场景适配：
- `之前写推文要1小时，现在5分钟出成品，差的不是速度是流程。`
- `从被女巫团灭到空投入账5位数，只搞懂了一个检测逻辑。`

### 4. 圈定受众模型 —— 精准点名 + "这篇是给你的"

原理：被点名的人天然会停下来看。

公式：`[所有XX的人/做XX的人]，[这篇你用得着]`

示例：
- `所有用AI写中文内容的，这篇规则可以直接丢给Agent用。`
- `所有不敢在链上冲土狗的，先把这3个工具装好。`
- `所有自己跑AI Agent的人，这个内存管理技巧能省一半token。`

AI/工具/Crypto 场景适配：
- `所有觉得"AI写的东西太AI了"的人，问题不在模型在规则。`
- `所有被mempool抢跑过的，这个工具专门治这个。`

### 5. 焦虑模型 —— 指出潜在危机 + 立刻缓解

原理：制造轻微不适感，然后给安全感。

公式：`[你可能正在错过/落后/犯错的信号]，[但来得及]`

示例：
- `AI要取代你的工作？但这3类人绝对安全…`
- `你的竞争对手可能已经在用AI批量写内容了，但你可以更快。`
- `空投越来越卷了，再不升级策略下一批就是陪跑。`

AI/工具/Crypto 场景适配：
- `AI Agent已经开始自主交易了，还在手动盯盘的人会被甩开。`
- `不会用AI做内容的人，下个季度会被会用的同事拉开差距。`

### 6. 恐惧模型 —— 放大坏结果 + 给预防方法

原理：比焦虑更强——直接描述损失场景，然后给盾牌。

公式：`[不做X的可怕后果]，[现在做Y可以避免]`

示例：
- `你的私钥可能已经暴露了，检查这3个地方立刻补救。`
- `不检查这5条规则，你的推文会继续被算法压下去。`
- `老板要裁人，第一个找的就是埋头苦干不会展示的人。`

AI/工具/Crypto 场景适配：
- `你的API key可能正在被薅，查一下用量记录就能发现。`
- `不改这个prompt习惯，每次生成都在浪费token。`

### 7. 高价值模型 —— "别处没有的独家干货"

原理：制造信息差优势感，让人想收藏。

公式：`[独家/内行才知道的] + [具体信息预告]`

示例：
- `分享一套Agent自动写推文后自己检查的规则，全网应该没有第二份。`
- `韩国化妆师绝不外传的底妆技巧，我替你整理好了。`
- `跑了一个月的AI Agent交易数据，总结出这3条反直觉结论。`

AI/工具/Crypto 场景适配：
- `我把human-social-copy所有规则跑了一遍对抗测试，这3条最容易被漏掉。`
- `整理了AI Agent大赛的薅毛清单，交易类/打德扑/打辩论的都有。`

### 8. 借势模型 —— "XX明星/专家也在用/说"

原理：权威背书降低信任成本。

公式：`[权威人物/热点事件] + [跟你要说的内容关联]`

示例：
- `Karpathy在最新视频里提到的方法，其实这个开源工具已经实现了。`
- `Vitalik刚聊完账户抽象，这个钱包已经把方案落地了。`
- `OpenAI刚发的那个新能力，这个开源项目半年前就在做了。`

AI/工具/Crypto 场景适配：
- `Sam Altman说AI Agent是下一波，其实已经有可直接部署的开源方案了。`
- `a16z刚投的那个赛道，这个项目比他们早走了半年。`

### 9. 反常识模型 —— 挑战"XX只能YY"的普遍假设

原理：打破读者的默认认知，制造"原来还能这样"的惊喜感。

公式：`[不是吧/居然]，[XX还能YY]？[事实是ZZ]`

跟好奇模型的区别：好奇模型是"只说一半"，反常识模型是"直接推翻一个你默认相信的东西"。跟反差模型的区别：反差是前后对比，反常识是横向颠覆——不是"以前差现在好"，而是"你以为只能A，其实还能B"。

示例：
- `不是吧，Codex还能用DeepSeek跑？`
- `原来不做动画的人也能用手绘+工具生成短片`
- `Codex不是OpenAI亲儿子吗，怎么接上Claude了`

AI/工具/Crypto 场景适配：
- `不是吧，AI Agent已经能自己写合约部署了？`
- `原来不写代码也能搭自动化工作流，拖拽就行`

## 钩子模板库

选好模型后，从下面的模板库里匹配具体句式。**这些是已验证的高互动句式，不要原创钩子结构——从库里选最接近的，填入具体信息。**

### 情绪冲击类（适合好奇/痛点/反差模型）

- `救命，[X]也太[形容词]了吧`
- `被问了N次的[X]，今天公开`
- `让我心心念念的[X]，终于[动词]了`
- `有被[X]惊艳到`
- `被夸爆的[X]，核心就一句话`
- `感觉以前的[X]都白[做]了`
- `后悔没有早点知道[X]`
- `我怎么没早点发现[X]`

### 工具/实用类（适合高价值/痛点模型）

- `[X]这个方法原来可以这样用`
- `10级懒人必备的[X]`
- `有了[X]，没人能比你早下班`
- `有手就会，新手小白也能[X]`
- `保姆级[X]教程，保证教会`
- `压箱底的[X]绝招，拿去吧`
- `你不知道的[X]隐藏用法`
- `[X]的正确打开方式`

### 反常识类（适合好奇/反差模型）

- `不是吧，你[X]还要花钱？`
- `不会还有人不知道[X]吧`
- `原来[X]才是真正的[Y]神器`
- `居然有人说[X]，真的会[Y]`
- `离谱，[X]这样做真的绝`
- `同样的[X]，为什么[Y]比你厉害？`

### 人群点名类（适合圈定受众模型）

- `所有[X]的人，请收藏这篇`
- `如果你是[X]，这3条规则会救你`
- `[X]星人本命，我真的太爱了`
- `通勤党/打工人/新手 本命[X]`
- `大数据请把这篇推给所有[X]的人`

### 结果承诺类（适合反差/焦虑模型）

- `看完这[X]你就[Y]了`
- `从0到1，我只[花了/用了] [X]`
- `从[X]到[Y]，我只[做了Z]`
- `学会这3点，保证让你[X]`
- `每天[X]分钟就能[Y]`

### 亲测/实证类（适合高价值/痛点/反差模型）

**这是让文案像真人的核心分类。** 不说"这个工具很好"，说"我用了X周/跑了X遍/截图为证"。

- `亲测有效，[X]确实能[Y]`
- `截图为证，[X]真的[做到了Y]`
- `用了[X]周/月，回测比我想的准`
- `跟风装了[X]，结果[Y]`
- `吃了/试了不下[X]遍，结论就一个`
- `跑了一轮[X]，发现最[形容词]的居然是[Y]`

### 稀缺/独家类（适合好奇/高价值模型）

- `99%的人不知道[X]可以[Y]`
- `我不允许你还不知道[X]`
- `[X]的正确打开方式，大部分人搞反了`
- `只有少数人知道的[X]隐藏用法`
- `[X]内部才用的[Y]方法，公开了`

### 平替/省钱类（适合痛点/圈定受众模型）

- `平替找到了！[X]完全可以替代[Y]，还免费`
- `[X]居然只要[价格]，之前白花[Y]了`
- `别再花[钱]买[X]了，这个开源版一样好用`
- `[大牌产品]的免费替代方案，效果不输`

## 爆款案例拆解：反常识 + 清单型钩子

以下是一条Crypto赛道的高互动长文拆解，展示钩子如何跟正文结构配合。

### 原文结构

```
钩子：薅毛死了？并没有！
      ↑ 反常识模型："大家都说X死了？其实没有"

重定义：只是用「AI Agent大赛」换了个名字给你送米
        ↑ 立刻重构认知，给读者一个"原来如此"的满足感

价值预告：整理了一份薅毛清单（交易/德扑/辩论都有）
          ↑ 具体可预期的价值，不是"我分析一下"而是"我整理了清单"

CTA前置：先马后看 ← 促进收藏

正文：【交易类】
      CREAO AI Agent交易比赛
      奖池$10,000，第一$3,000
      模拟盘500U，不用入金
      默认策略做空ETH（真的懂币圈）
      Agent跑7天
      1000名额先到先得
      ...更多类别...
```

### 可复用的结构

```
钩子：大家都说[X死了/不行了]？并没有！  ← 反常识模型
重定义：只是换了个形式在[做什么]            ← 立刻给新视角
清单预告：整理了[X类/Y类/Z类都有]          ← 具体价值
正文：分类 + 具体数字 + 降低门槛的细节     ← 干货
```

### 适用场景

- 某赛道被唱衰但其实有新机会（DeFi/空投/NFT/GitHub开源…）
- "XX已死"叙事反转
- 新玩法出现但大多数人还没注意到

这个结构的核心：**钩子挑起情绪 → 立刻反转给新视角 → 马上证明自己有价值（清单）→ 正文全是干货。** 不要先分析为什么没死，直接给清单。

钩子不止是第一行。三个黄金位置决定帖子生死：

1. **第一句（钩子本体）**—— 决定刷到的人停不停
2. **正文第一句**—— 决定停下来的人读不读
3. **真诚是底牌**—— 钩子吸引进来，实料才能让人留下

钩子可以组合使用：比如"圈定受众 + 好奇模型"→ `所有用AI写推文的人，我发现了让AI不再像AI的规则——核心不是prompt...`

## 快速自检

写完钩子后问自己：

- 读者第一眼能不能知道这条内容和自己有什么关系
- **第一行有没有用到一个具体模型（好奇/痛点/反差/圈定/焦虑/恐惧/高价值/借势/反常识），还是写了一个空泛判断**
- 有没有从模板库里选了最接近的句式，还是自己编了一个没验证过的结构
- 有没有为了显得有观点而写空话
- 有没有把真正有用的信息藏到后面

## 不要用的钩子

- 空泛判断句：`AI工具真的越来越强了。`
- 标题党式提问：`你知道AI会取代你吗？` （没有给信息，只制造不适）
- 先评价再解释：`这是个很有意思的项目，它...`
- 大词开头：`在AGI时代...` / `随着区块链技术发展...`
- 没有具体对象的提醒：`注意了！AI领域又有大事发生...`
