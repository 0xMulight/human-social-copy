# 个人语气采集Prompt

把这段Prompt复制给任何Agent，用来从你的旧内容里提取个人语气画像。

```text
你是我的个人语气分析Agent。

我会给你几篇我自己写过的内容。你的任务不是改写，而是提取我的表达方式。

请按下面结构输出：

1. 语气定位
用一句话说明我的写作给人的感觉。

2. 读者对象
判断我主要写给谁看。

3. 常用开头
总结我常用的开头方式，并给5个可复用句式。

4. 常用正文结构
总结我展开内容的方式，比如先提醒、再拆重点、最后给观察顺序。

5. 常用句式
提取我高频使用的句型。不要创造新风格，只总结已有风格。

6. 节奏和排版
总结我的句子长度、段落长度、小标题使用方式、数字清单使用方式。

7. 风险表达
总结我怎么表达不确定性、风险、免责声明和边界。

8. 互动结尾
总结我怎么自然引导评论、收藏或讨论。

9. 禁止偏移
列出10条“写得不像我”的表达方式。

10. 可直接使用的系统提示词
把上面的分析压缩成一段可复制的系统提示词，供下次生成内容时使用。

要求：
- 不要夸我
- 不要写空泛评价
- 不要模仿其他博主
- 不要只说“简洁、自然、有洞察”这种空话
- 必须给具体句式和判断标准
- 保留我的语气，但不要保留错别字或明显病句
```

## 使用方式

第一步，准备5到10篇自己满意的旧内容。

第二步，把这些内容一起发给Agent，再粘贴上面的Prompt。

第三步，让Agent生成个人语气画像。

第四步，把语气画像保存下来。以后改写内容时，先加载这份语气画像，再加载human-social-copy规则。

## 生成内容时的调用Prompt

```text
请按我的个人语气画像和human-social-copy规则，改写下面这段内容。

优先级：
1. 事实准确
2. 我的语气
3. 内容结构
4. 平台适配
5. 传播性

要求：
- 不要写成财经媒体通稿
- 不要写成AI总结稿
- 不要过度夸张
- 保留我常用的提醒式表达
- 保留我的短句节奏
- 每段都要有具体信息
- 结尾用自然互动或学习记录收尾
```
